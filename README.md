## ⬇️ How to Install

1. Click the **Download** button above (or go to [Releases](https://github.com/Vem-Manish/Laptop-Gesture-Control-App/releases/latest))
2. Download `Hand.Gesture.Control.exe`
3. Double-click it — done ✅

> No Python. No setup. No installer. Just run the EXE.  
> The app starts silently in your system tray and **auto-starts on every boot** automatically.

---

## 🎮 How to Use

| Step | Action |
|---|---|
| **1** | Run the EXE — it appears in your system tray (bottom-right) |
| **2** | Press **`Ctrl + G`** to open the control panel |
| **3** | Click **Start** and point your hand at the webcam |
| **4** | To quit, right-click the tray icon → **Exit** |

---

## 🖐️ Gestures

| Hand Gesture | What it does |
|---|---|
| ☝️ 1 finger | Move mouse cursor |
| ☝️ Pinch with thumb | Left click / Click & hold |
| ✌️ 2 fingers | Scroll up / down |
| 🤟 3 fingers swipe up | Task View (`Win + Tab`) |
| 🤟 3 fingers swipe down | Minimize window |
| ✋ 4 fingers swipe right | Switch app (`Alt + Tab`) |
| ✋ 4 fingers swipe left | Switch back (`Alt + Shift + Tab`) |

## 🐛 Troubleshooting

**App opened but Start does nothing**
→ Make sure your webcam is connected and not being used by another app (Teams, Zoom, etc.)

**Gestures are jittery or inaccurate**
→ Make sure you have good lighting. Dark rooms reduce detection quality.

**Tray icon not showing**
→ Check the hidden icons arrow (^) in your taskbar bottom-right corner.

**App not auto-starting after reboot**
→ Run the EXE once manually — it registers itself on first launch.
